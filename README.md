**Date :** Juillet 2026

## Description
Site vitrine de la conférence tech panafricaine AfriConnect Summit 2026.

## Technologies utilisées
- HTML5 sémantique
- CSS3 (Flexbox, Grid, Variables CSS, Animations, Dark Mode)
- JavaScript Vanilla
- Git & GitHub Pages

## Fonctionnalités JavaScript
1. Dark Mode / Light Mode avec localStorage
2. Navbar dynamique (scroll effect à 80px)
3. Menu hamburger responsive
4. Compte à rebours temps réel
5. Compteurs animés au scroll
6. Animations fade-in/slide-in au scroll
7. Onglets du programme (3 jours)
8. Filtrage dynamique des intervenants
9. Validation de formulaire
10. Bouton retour en haut
11. Année dynamique dans le footer
